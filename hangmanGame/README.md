<strong>Juego del Ahorcado </strong>

Código del típico juego del ahorcado en donde con manipulación de archivos y el método random selecciona una palabra y luego con ayuda del método input se puede comparar las sílabas de la palabra elegida y la sílabas que usuario ingrese genere un resultado. Lenguaje Python. 

Para contribuir puedes crear un pull request al proyecto.
